Cascading StyleSheets (CSS) - Anatomy of a Style

In CSS there are three ways to apply styles to an HTML page or pages: (1) inline styles, (2) internal styles, and (3) style sheets. All these ways use CSS Rules which are statements about a particular stylistic aspect of an element or elements.

Example of a Rule

A Ruleconsists of two parts: Selector - that part before the left curly brace, and Declaration - that part within the curly braces

h1 { color:red }

Examples of a Declaration

A declaration has two parts separated by a colon: Property - that part before the colon Value - that part after the colon

color:red

Grouping Selectors

Group the selectors into a comma-separated list and only write the declaration

  • h1 { font-weight: bold }
  • h2 { font-weight: bold }
  • h3 { font-weight: bold } becomes:
  • h1, h2, h3 { font-weight: bold }

Grouping Declarations

CSS rules often have more that one declaration

Group declarations that relate to the same selector into a semicolon-separated list

  • h1 { color: red }
  • h1 ( text-align: center } becomes:
  • h1 { color: red; text-align: center; }
